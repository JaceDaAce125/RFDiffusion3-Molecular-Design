# RFDiffusion3-Molecular-Design
RFDiffusion3 for generation of de novo protein structures with AI assistance.

Created by Rosetta in the University of Washington, RFD3 creates de novo proteins for specificity, catalysis, and binding with protein on protein, DNA, and small molecules. This model facilitates atom-level diffusion, and allows researchers to design exact atomic interactions, making the program more flexible and the product more exact. 

## Workflow
1. Set up environment (Python, Torch)
2. Install RFdiffusion
3. Run generation scripts
4. Analyze outputs
